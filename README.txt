LINUX STEPS TO INSTALL AND RUN
1. Clone or Download zip file
2. Unzip files
3. Open a terminal and navigate to the folder of the unziped; typically the Downloads folder 
3. run the command 'python ElctrcFldspy.py'

2. Unzip files
